Coyote Game Hub

郊狼战败惩罚控制器

开始使用资源下载

DG-Lab Socket

通过DG-Lab Socket，您可以快速且稳定地连接到郊狼3.0设备。

HTTP API

通过HTTP API，您可以使用任何编程语言快速设置郊狼设备电量。

蓝牙直连

可以在浏览器中直接连接到郊狼设备，无需安装任何插件，经过特殊的优化，延迟极低。

郊狼2.0设备

即使是2.0设备，也可以通过蓝牙进行连接，通过此程序可以控制几乎所有的郊狼设备。

电量显示小组件

控制器内提供了一个小组件，可以在任何支持Webview的位置显示设备电量。

更多扩展插件

基于此控制器，可以开发更多有趣的插件。我们已经开发了一些插件，您可以在资源下载中找到。
